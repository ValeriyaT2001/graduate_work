Выберите правильные опрделения существенной и фиктивной переменных
Переменная xi фиктивная
Переменная xi существенная

Выполняется условие: f(x1, ..., xi-1, 0, xi+1, ..., xn) ≠ f(x1, ..., xi-1, 1, xi+1, ..., xn)

Выполняется условие: f(x1, ..., xi-1, 0, xi+1, ..., xn) = f(x1, ..., xi-1, 1, xi+1, ..., xn)
